# Understanding Caveman

Cavemanized documents are compact machine-facing instructions: terse bullets, labels, and arrow flows still carry the same priority as normal prose.
Missing explanation is intentional; follow explicit rules, `MUST`/`NEVER` gates, commands, paths, URLs, code, and ordered flows exactly.
